# Demo Script

## 60-Second Pitch

MagneticSphere AI is an autonomous opportunity discovery platform. Instead of waiting for reps to manually research accounts, specialized AI agents monitor funding, hiring, news, tech stack, and social sentiment signals. The reasoning layer combines those signals with graph and vector memory, predicts which companies are likely to buy, explains why, and triggers workflows like email, Slack alerts, and CRM updates.

## Walkthrough

1. Start on the dashboard and show the ranked opportunities.
2. Select `NeuroTech AI`.
3. Explain the score:
   - Series A funding
   - Hiring ML engineers
   - Positive AI adoption sentiment
   - Strong tech-stack match
4. Show the graph view to highlight GraphRAG reasoning.
5. Click or show generated outreach email.
6. Point out workflow actions: Slack alert, CRM update, meeting recommendation.

## Judge-Friendly Differentiators

- Multi-agent architecture instead of a single prompt.
- Explainable predictive scores instead of static CRM lead scores.
- GraphRAG plus vector memory for relationship-aware reasoning.
- Automated action layer, not just analytics.
